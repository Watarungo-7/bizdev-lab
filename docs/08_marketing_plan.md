# マーケティング計画: Desk Nest Cat Bed 日本独占販売事業

## エグゼクティブサマリー

「ユーザー1人 + AI」体制で実行可能な集客・ブランド戦略を策定。Phase Aでは広告テスト（¥14,000）でWTP・CACを検証し、Phase BではMakuakeプロジェクトの初動をSNS・インフルエンサー・PRで盛り上げる。Phase CではAmazon JP・自社ECへの継続的な集客を実現。AI活用によりSNS投稿・広告クリエイティブ・コンテンツ生成を自動化し、低予算・高効率のマーケティングを実現する。

## 1. ターゲット顧客 & メッセージング

### コアメッセージ

**ワンライナー**: 「猫のキーボード占領問題を解決する、デスク取付型猫ベッド」

**エレベーターピッチ（30秒）**: 
「リモートワーク中、猫がキーボードの上に座って作業を妨害する経験はありませんか？Desk Nestは、デスク端に取り付ける専用ベッドで、猫を近くに置きながら作業スペースを確保できます。世界初のデスク取付型猫ベッドで、Kickstarterで$468Kの支援を集めた実績があります。」

**顧客の言葉で語る課題**: 
「猫を追い払いたくないけど、作業効率も確保したい」「デスク上で猫に快適に過ごしてほしい」「既存のベッドではデスク上に配置できない」

### メッセージングマトリクス

| ペルソナ | 課題 | 提供価値 | 響くメッセージ |

### 各チャネルの詳細施策

#### SNS（Twitter/X）

**目的**: 「ニャンサムウェア」キーワードでの拡散、課題の共感獲得

**施策**:
- 日次投稿（1日2-3回）: 課題の共感、製品紹介、使用シーン
- ハッシュタグ活用: #ニャンサムウェア #リモートワーク #猫用品 #デスクワーク
- UGC促進: 顧客の使用シーン投稿をリツイート・紹介
- インフルエンサーとのコラボ: 猫アカウント・リモートワーカーアカウントとの連携

**AI自動化**:
- 投稿内容の生成（Claude/GPT-4）
- ハッシュタグの最適化
- エンゲージメント分析

**目標**: Phase Bでフォロワー500人以上、Phase Cで1,000人以上

#### SNS（Instagram）

**目的**: ビジュアル訴求、UGC促進、ブランド認知拡大

**施策**:
- 週次投稿（週3-5回）: 製品写真、使用シーン、猫の可愛い瞬間
- Reels活用: 15-30秒の短尺動画で使用シーンを紹介
- ハッシュタグキャンペーン: #うちの猫デスクベストショット 等
- ストーリーズ: 日常的な使用シーン、限定情報

**AI自動化**:
- 投稿内容の生成（Claude/GPT-4）
- キャプションの最適化
- ハッシュタグの最適化

**目標**: Phase Bでフォロワー300人以上、Phase Cで500人以上

#### Makuake（プラットフォーム内）

**目的**: プロジェクトページへの集客、応援購入の獲得

**施策**:
- プロジェクトページの最適化: 写真・動画・ストーリーの充実
- 早割・限定リターンの設定: 初動を盛り上げる
- 応援コメントへの返信: コミュニティ形成
- 活動レポートの定期更新: 進捗共有による信頼獲得

**AI自動化**:
- プロジェクトページの下書き作成（Claude/GPT-4）
- 活動レポートの生成
- 応援コメントへの返信（FAQ対応）

**目標**: Phase Bで100件以上の応援購入達成

#### インフルエンサーマーケティング

**目的**: 信頼性の獲得、拡散力の向上

**施策**:
- マイクロインフルエンサー（フォロワー1,000-10,000人）へのサンプル提供
- 猫アカウント・リモートワーカーアカウントとのコラボ
- 実機レビュー動画の制作依頼

**対象**:
- 猫専門アカウント（Instagram、Twitter）
- リモートワーク・デスクワーク関連アカウント
- ペット用品レビューアカウント

**予算**: Phase Bで¥50,000-100,000（サンプル提供 + 謝礼）

#### リスティング広告（Google Ads）

**目的**: 検索需要の獲得、LPへの集客

**施策**:
- 検索広告: 「デスク 猫ベッド」「猫 キーボード 対策」等のキーワード
- ディスプレイ広告: リマーケティング
- YouTube広告: 製品紹介動画の配信

**AI自動化**:
- 広告コピーの生成（Claude/GPT-4）
- キーワードの最適化
- パフォーマンス分析

**予算**: Phase Bで¥50,000-100,000、Phase Cで¥100,000-200,000/月

#### SNS広告（Meta Ads）

**目的**: ターゲティング広告による集客

**施策**:
- Facebook広告: 25-40代、猫飼い、リモートワーク関連の興味関心でターゲティング
- Instagram広告: ビジュアル訴求
- 動画広告: 製品紹介動画の配信

**AI自動化**:
- 広告クリエイティブの生成（Claude/GPT-4 + DALL-E）
- ターゲティング設定の最適化
- パフォーマンス分析

**予算**: Phase Bで¥50,000-100,000、Phase Cで¥100,000-200,000/月

#### PR・プレスリリース

**目的**: メディア掲載による認知拡大

**施策**:
- プレスリリース配信: Makuakeプロジェクト開始時、成功時
- メディアピッチ: ペット関連メディア、デスクワーク関連メディアへの個別アプローチ
- ブロガー・ライターへのサンプル提供

**AI自動化**:
- プレスリリースの作成（Claude/GPT-4）
- メディアリストの作成
- ピッチメールの作成

**予算**: Phase Bで¥0-10,000（配信サービス利用料）
