# 顧客検証レポート: Desk Nest Cat Bed 日本独占販売事業

## エグゼクティブサマリー

ターゲット顧客（猫飼い×リモートワーカー20〜40代）は明確に定義可能で、SNS上で「猫のキーボード占領問題」に関する投稿が多数存在し、課題の深刻度は高い（Pain Level: 8/10推定）。ただし、WTP（支払意思）については¥12,800-16,800の価格帯での検証が必要。既存の代替手段（窓取付ハンモック、デスク下ベッド）では課題が十分に解決されておらず、Desk Nestの差別化が明確。

## 1. ペルソナ定義

### ペルソナ A: リモートワーカー×猫飼い（プライマリ）

**デモグラフィック**:
- 年齢: 25-40代
- 性別: 男女問わず（やや女性比率が高い可能性）
- 職業: IT・クリエイティブ・コンサル・金融等の知識労働者
- 年収: 400-800万円
- 居住地: 都市部マンション（1LDK-2LDK）

**サイコグラフィック**:
- 価値観: ワークライフバランス重視、ペットを家族として扱う、生活の質（QoL）への投資を惜しまない
- ライフスタイル: 在宅勤務中心、デスクワーク時間が長い、SNSでペットの写真を投稿する
- 情報収集手段: Twitter/X、Instagram、YouTube、Amazonレビュー、Makuake

**行動特性**:
- 購買行動: オンライン購入が中心、レビューを重視、クラウドファンディングにも参加経験あり
- 意思決定プロセス: 課題解決の必要性を感じる → 情報収集（レビュー・SNS） → 比較検討 → 購入

**課題・ペイン**:
- 猫がキーボードの上に座って作業を妨害する
- 猫を追い払うとストレスを感じるが、作業効率も下がる
- 猫に近くにいてほしいが、作業スペースを確保したい
- 既存の猫ベッドではデスク上に配置できない

**現在の解決手段**:
- 猫を別の部屋に閉じ込める（ストレス）
- キーボードを物理的に隠す（不便）
- 猫用の箱やベッドをデスクの近くに置く（効果が限定的）
- 諦めて作業効率の低下を受け入れる

**理想の状態**:
- 猫がデスク上で快適に過ごせる
- 作業スペースを確保できる
- 猫との距離感が適切で、ストレスなく作業できる
- 見た目もおしゃれで、デスク環境に馴染む

### ペルソナ B: ゲーマー×猫飼い（セカンダリ）

**デモグラフィック**:
- 年齢: 20-35代
- 職業: 学生、フリーランス、ゲーム実況者等
- 年収: 200-600万円
- 居住地: 都市部・地方問わず

**サイコグラフィック**:
- 価値観: ゲームとペットの両立、快適なゲーム環境への投資
- ライフスタイル: 長時間デスクに座る、ゲーム配信を行う、SNSでゲーム・ペットの情報を発信

**課題・ペイン**:
- ゲーム中に猫がキーボードやマウスを占領する
- 配信中に猫が画面に入ってくる
- ゲーム環境を整えたいが、猫の居場所も確保したい

### ペルソナ C: 多頭飼い世帯（セカンダリ）

**デモグラフィック**:
- 年齢: 30-50代
- 職業: 様々
- 年収: 500万円以上
- 居住地: 都市部・地方問わず

**課題・ペイン**:
- 複数の猫がデスク周りに集まる
- 1つのベッドでは足りない
- 各猫に快適な居場所を提供したい

## 4. 支払意思（WTP）分析

| 分析項目 | 結果 |
|---------|------|
| **類似サービスの価格帯** | ¥3,500-4,500（窓取付ハンモック）、¥5,000-10,000（デスク下ベッド）、¥12,000-16,000（自動給餌器・給水機） |
| **顧客の価格許容範囲（推定）** | ¥10,000-20,000（ペット用品の高機能製品の価格帯を参照） |
| **価値の定量化** | 作業効率向上による時間価値: 1日30分の作業時間確保 × 時給3,000円 × 20営業日 = 月間¥18,000相当の価値 |
| **スイッチングコスト** | 低（既存の解決手段が不十分なため、新規購入の障壁は低い） |

**推奨価格帯**: ¥12,800-16,800（Makuake応援購入価格）

**WTP検証の必要性**:
- 間接競合（¥3,500-4,500）と比較して約3-4倍の価格差がある
- 自動給餌器・給水機（¥12,000-16,000）と同価格帯だが、機能が異なる
- 実際のWTPは広告テスト（¥14,000）で検証が必要
