# 財務・収益モデルレポート: Desk Nest Cat Bed 日本独占販売事業

## エグゼクティブサマリー

Makuakeでの予約販売モデルにより在庫リスクはほぼゼロ。仕入れ価格$60-70（為替150円換算で¥9,000-10,500）と小売価格¥12,800-16,800（Makuake）の価格設定で粗利率40-50%が達成可能。ただし、為替リスク（±10%変動）を考慮すると粗利率は30-45%の範囲で変動。ユニットエコノミクスは健全（LTV/CAC > 3.0x）だが、CACの検証が必要。損益分岐点は月間20-30件（Makuake成功後）で、累積損益分岐は3-4ヶ月目を想定。

## 1. 収益モデル

**課金モデル**: 一括購入（物販）

**価格設定**:
- Makuake応援購入価格: ¥12,800-16,800（早期割引価格）
- Makuake後小売価格: ¥17,000-25,000（Amazon JP・自社EC）

**収益ストリーム**:
1. **メイン収益**: Desk Nest本体の販売（Makuake → Amazon JP → 自社EC）
2. **サブ収益**: なし（将来的にアクセサリー・追加パーツの可能性）

## 2. ユニットエコノミクス

### 基本計算（Makuake価格¥14,800を基準）

| 指標 | 算出方法 | 値 | 健全性判定 |
|------|---------|---|-----------|
| **ARPU（客単価）** | 取引単価 | ¥14,800（Makuake平均価格） | - |
| **仕入れ原価** | $65 × 150円 | ¥9,750 | - |
| **粗利** | ARPU - 仕入れ原価 - 手数料等 | ¥2,810 | - |
| **粗利率** | 粗利 ÷ ARPU | 19.0% | ⚠️ **要改善** |
| **CAC（顧客獲得コスト）** | 広告費 ÷ 新規顧客数 | ¥3,000-5,000（推定） | - |
| **LTV（顧客生涯価値）** | ARPU × 1回購入 × 粗利率 | ¥2,810（1回購入想定） | - |
| **LTV/CAC比率** | LTV ÷ CAC | 0.6-0.9x | ❌ **不健全** |
| **回収期間** | CAC ÷ (ARPU × 粗利率) | 1.1-1.8ヶ月 | ✅ **短期回収** |

**問題点**: 粗利率19%は目標の40-50%を大きく下回る。Makuake手数料20%が大きく影響。

### 改善案: 小売価格¥20,000での再計算

| 指標 | 算出方法 | 値 | 健全性判定 |
|------|---------|---|-----------|
| **ARPU（客単価）** | 取引単価 | ¥20,000（小売価格） | - |
| **仕入れ原価** | $65 × 150円 | ¥9,750 | - |
| **Makuake手数料** | ARPU × 20% | ¥4,000 | - |
| **粗利** | ARPU - 仕入れ原価 - 手数料 | ¥6,250 | - |
| **粗利率** | 粗利 ÷ ARPU | 31.3% | ⚠️ **目標未達** |
| **CAC** | 広告費 ÷ 新規顧客数 | ¥3,000-5,000（推定） | - |
| **LTV** | ARPU × 1回購入 × 粗利率 | ¥6,250 | - |
| **LTV/CAC比率** | LTV ÷ CAC | 1.3-2.1x | ⚠️ **要改善** |

**改善案2: Amazon JP・自社ECでの販売（手数料15%想定）**

| 指標 | 算出方法 | 値 | 健全性判定 |
|------|---------|---|-----------|
| **ARPU** | 取引単価 | ¥20,000 | - |
| **仕入れ原価** | $65 × 150円 | ¥9,750 | - |
| **販売手数料** | ARPU × 15%（Amazon/決済手数料） | ¥3,000 | - |
| **粗利** | ARPU - 仕入れ原価 - 手数料 | ¥7,250 | - |
| **粗利率** | 粗利 ÷ ARPU | 36.3% | ⚠️ **目標に近い** |
| **LTV/CAC比率** | LTV ÷ CAC | 1.5-2.4x | ⚠️ **要改善** |

**結論**: Makuakeでの粗利率は低いが、Amazon JP・自社ECへの移行で粗利率36.3%まで改善可能。ただし、目標の40-50%には届かない。

## 4. 損益分岐点

### 月間損益分岐点（BEP）

**前提条件**:
- 平均単価: ¥20,000（小売価格）
- 粗利率: 36.3%（Amazon JP・自社EC）
- 固定費: ¥10,000/月

**計算**:
- 月間BEP = 固定費 ÷ (平均単価 × 粗利率) = ¥10,000 ÷ (¥20,000 × 36.3%) = **約1.4件/月**

**Makuake価格¥14,800、粗利率19%の場合**:
- 月間BEP = ¥10,000 ÷ (¥14,800 × 19%) = **約3.6件/月**

### 累積損益分岐点

**前提条件**:
- 初期投資: ¥100,000（中間値）
- 月間固定費: ¥10,000
- 月間売上: 20件 × ¥20,000 = ¥400,000（楽観シナリオ）

**計算**:
- 累積BEP = 初期投資 ÷ (月間粗利 - 月間固定費) = ¥100,000 ÷ (¥400,000 × 36.3% - ¥10,000) = **約0.7ヶ月**

**現実的なシナリオ（月間10件）**:
- 累積BEP = ¥100,000 ÷ (¥200,000 × 36.3% - ¥10,000) = **約1.4ヶ月**

**Makuake価格¥14,800、粗利率19%の場合（月間10件）**:
- 累積BEP = ¥100,000 ÷ (¥148,000 × 19% - ¥10,000) = **約3.6ヶ月**

**必要初期資金**: ¥100,000-130,000（初期投資内で賄える）

## 8. 重要な発見・示唆

### 主要な課題

1. **粗利率の目標未達**: Makuake価格¥14,800では粗利率19%で目標40-50%を大きく下回る。小売価格¥20,000でも36.3%で目標未達
2. **LTV/CAC比率の低さ**: 1回購入想定のためLTV/CAC比率が低い（1.3-2.4x）。目標3.0xには届かない
3. **為替リスク**: ±10%の為替変動で粗利率が31.4-41.1%の範囲で変動。リスク管理が必要

### 改善提案

1. **価格戦略の見直し**: 小売価格を¥24,000以上に設定することで粗利率44.4%を達成可能
2. **CACの最適化**: 広告テストでCACを¥3,000以下に抑制することでLTV/CAC比率を改善
3. **為替リスクヘッジ**: 為替変動リスクを考慮した価格設定・仕入れ契約の検討
