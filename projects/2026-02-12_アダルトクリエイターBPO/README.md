# アダルトクリエイターBPO（AI運用代行サービス）

## このフォルダについて

本事業の仮説検証・実行計画の成果物を格納するプロジェクトフォルダです。

- **入力**: `input.md`（事業概要入力シート）
- **Phase 1 成果物**: 01_hypothesis.md ～ 06_go_nogo_verdict.md（仮説検証チームの出力）
- **Phase 2 成果物**: 07_business_roadmap.md ～ 09_sales_plan.md（実行計画）
- **その他**: エグゼクティブサマリー・計画書・財務・KPI・タスク等は検証完了後に追加

## 進め方

1. Phase 1: `input.md` を元に①仮説設計 → ②市場・競合調査 → ③顧客検証 → ④財務 → ⑤リスク → ⑥統合判定
2. Go 判定 + 承認後: Phase 2 で⑦⑧⑨の実行計画を策定
3. 成果物はすべてこのフォルダ内に出力する
