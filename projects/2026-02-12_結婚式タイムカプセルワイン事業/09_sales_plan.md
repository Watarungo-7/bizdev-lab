# 営業・セールス計画: 結婚式タイムカプセルワイン事業

## エグゼクティブサマリー

本計画は「ユーザー1人 + AI」体制で実行可能なセールス戦略を策定したものである。Phase B（Makuake）ではクラウドファンディング型のセルフサーブ販売、Phase C（EC本格稼働）ではインバウンド営業とAI自動化、Phase D以降では式場提携・法人向けアウトバウンド営業へと段階的に拡大する。

**セールス戦略の核心**:
- **Phase B**: Makuakeプラットフォームの集客力を活用し、広告費ゼロで50-100名の初期顧客を獲得
- **Phase C**: 自社ECでの自動化セールスファネルを構築し、CAC 2,500円以下を維持
- **Phase D**: 式場提携で新郎新婦への直接リーチを確保、法人周年記念で高単価案件を獲得

**AI自動化目標**:
| フェーズ | AI自動化率 | ユーザー対応 |
|---------|-----------|-------------|
| Phase B | 40% | Makuake運営、コメント対応 |
| Phase C | 70% | 受注発送、特殊対応 |
| Phase D | 60% | 法人営業、式場交渉 |

**収益目標**:
| 指標 | Phase B | Phase C (Y1) | Phase D (Y2) |
|------|---------|-------------|-------------|
| 販売本数 | 50本 | 70本 | 250本 |
| 売上 | 60万円 | 84万円 | 300万円 |
| 顧客数 | 40名 | 55名 | 150名 |

## 1. セールスモデル

### フェーズ別セールスモデル

| フェーズ | モデル | 特徴 | 顧客獲得チャネル |
|---------|--------|------|-----------------|
| **Phase B（Makuake）** | クラウドファンディング型 | プラットフォーム内流入中心、手数料20% | Makuake、SNS告知 |
| **Phase C（EC本格）** | セルフサーブ + インバウンド | 自社EC、広告・SEO経由、決済手数料4% | Instagram、Google、SEO |
| **Phase D（式場・法人）** | アウトバウンド + パートナー | 提携営業、法人直接営業 | 紹介、営業活動 |

### 各モデルの詳細

#### Phase B: Makuakeモデル
```
[Makuakeユーザー]
     ↓ プラットフォーム内検索・閲覧
[プロジェクトページ閲覧]
     ↓ ストーリー・リターン確認
[応援購入（先払い）]
     ↓ Makuake経由決済
[プロジェクト終了後]
     ↓ 商品制作・発送
[顧客データ取得]
     ↓ リマインドサービス登録
[リピート・紹介促進]
```

**Makuakeの強み**:
- 広告費ゼロで新規顧客にリーチ可能
- 「応援購入」マインドで価格弾力性が高い
- プレスリリース・メディア露出の機会

**注意点**:
- 手数料20%で粗利率が42%に低下
- 顧客データの直接取得に制限あり
- プロジェクト終了後のEC移行が必須

#### Phase C: 自社ECモデル
```
[SNS/検索/広告]
     ↓ 認知・興味
[LP/ECサイト訪問]
     ↓ 商品理解
[カート投入]
     ↓ 購入検討
[決済完了（先払い）]
     ↓ 受注確認メール（AI自動）
[制作・発送]
     ↓ 発送通知メール（AI自動）
[リマインドサービス登録]
     ↓ 記念日リマインド（AI自動）
[リピート購入/紹介]
```

**自社ECの強み**:
- 粗利率58%（Makuake比+16ポイント）
- 顧客データの完全所有
- リマインド配信による関係構築

#### Phase D: 法人・式場モデル
```
[ターゲットリスト作成]
     ↓ AI支援
[アウトリーチ（メール/DM）]
     ↓ AI下書き + ユーザー送信
[初回面談/提案]
     ↓ ユーザー対応
[見積もり・サンプル提供]
     ↓ AI見積書作成
[契約・受注]
     ↓
[納品・請求]
     ↓
[リピート提案]
```

### AI自動化率目標

| リマインド配信 | - | 100% | 100% |
| 営業リスト作成 | - | - | 90% |
| 提案書作成 | - | - | 80% |
| **合計自動化率** | **40%** | **70%** | **60%** |

## 5. 法人営業戦略（Phase D）

### 式場提携

#### ターゲット式場の選定基準

| 基準 | 優先度 | 理由 |
|------|--------|------|
| 挙式件数 年200件以上 | 高 | 一定のボリュームが見込める |
| 演出オプションへの積極性 | 高 | 新しいサービス導入に前向き |
| 独立系式場（チェーン以外） | 中 | 決裁スピードが速い |
| 首都圏・関西圏 | 中 | 初期は近距離で対応可能 |
| ナチュラル・ガーデン系 | 中 | ワインとの親和性高 |

#### 式場提携のアプローチ方法

**Step 1: リスト作成（AI支援）**
- ゼクシィ、ハナユメ等の式場情報から候補抽出
- 100式場をリスト化（所在地、規模、特徴）

**Step 2: アウトリーチ**
| 方法 | 優先度 | 理由 |
|------|--------|------|
| 式場紹介サービス経由 | 高 | 既存関係があり紹介しやすい |
| ウェディングプランナー直接 | 高 | 決裁権を持つケースあり |
| 式場代表メール | 中 | 担当者へ振り分けてもらう |
| 展示会・イベント | 低 | コスト高、優先度低 |

**Step 3: 提案内容**
| 提案項目 | 内容 |
|---------|------|
| 導入メリット | 他式場との差別化、単価向上、顧客満足度向上 |
| 収益モデル | 紹介手数料10-15% or 卸売価格での提供 |
| オペレーション | 式場負担ゼロ（パンフレット設置のみ） |
| サンプル提供 | 体験用サンプル2本無償提供 |
| 実績 | Makuake支援者数、EC販売実績 |

**Step 4: 契約条件**
| 項目 | 条件 |
|------|------|
| 契約形態 | 業務提携契約 |
| 手数料 | 紹介成約 15% or 卸売（10,000円/本） |
| 最低契約期間 | 6ヶ月 |
| 解約条件 | 30日前通知 |
| 独占/非独占 | 非独占（複数式場と提携可能） |

#### 式場提携のKPI

| KPI | Year 2 Q1 | Year 2 Q2 | Year 2 Q3 | Year 2 Q4 |
|-----|-----------|-----------|-----------|-----------|
| アプローチ数 | 20社 | 30社 | 30社 | 20社 |
| 面談数 | 3社 | 5社 | 5社 | 3社 |
| 提携成立 | 0社 | 1社 | 0社 | 0社 |
| 紹介成約数 | - | 2件 | 5件 | 8件 |

| セグメント | 特徴 | ロット規模 | 営業優先度 |
|-----------|------|-----------|-----------|
| 創業10/20/30周年企業 | 節目の年、記念品需要 | 30-100本 | 高 |

## 9. 撤退・ピボット基準

### 撤退基準

| フェーズ | 撤退条件 | 判断時期 | 累計損失上限 |
|---------|---------|---------|-------------|
| Phase B | Makuake達成率30%未満 | Day 150 | 20万円 |
| Phase B | 12,000円リターン選択率20%未満 | Day 150 | 20万円 |
| Phase C | 月間5本未満 × 3ヶ月連続 | Month 8 | 30万円 |
| Phase C | CAC 6,000円超 × 2ヶ月連続 | 随時 | その時点の累計 |
| Phase C | 累計100本未満 | Month 12 | 40万円 |
| Phase D | 法人成約0件（Year 2通年） | Year 2末 | 事業判断 |

### ピボット選択肢

| 状況 | ピボット選択肢 | 具体的アクション |
|------|---------------|-----------------|
| WTP 12,000円不成立 | 価格引き下げ | 10,000円で再テスト販売 |
| 個人向け不振 | 法人特化 | 式場・企業周年に集中 |
| 結婚式向け不振 | 記念日汎用化 | 還暦・退職・出産記念に拡大 |
| 国内ワイン確保困難 | 輸入ワイン特化 | インポーターと提携 |
| 式場提携不成立 | EC直販強化 | インフルエンサーマーケ強化 |
| ワイン事業困難 | サービス転換 | リマインドサービスのみSaaS化 |

### 撤退判断フローチャート

```
[Makuake終了]
     │
     ├─ 達成率70%以上 → Phase C（EC本格稼働）
     │
     ├─ 達成率30-69%
     │   └─ 価格10,000円で再テスト（3ヶ月）
     │       ├─ 月10本以上 → Phase C（価格10,000円）
     │       └─ 月10本未満 → 撤退
     │
     └─ 達成率30%未満 → 撤退 or 法人特化ピボット
             │
             └─ 法人特化の場合
                 ├─ 3ヶ月で法人問い合わせ5件以上 → 継続
                 └─ 3ヶ月で法人問い合わせ5件未満 → 撤退

[Phase C終了（Month 12）]
     │
     ├─ 累計100本以上 → Phase D（拡大期）
     │
     └─ 累計100本未満 → 撤退
```
