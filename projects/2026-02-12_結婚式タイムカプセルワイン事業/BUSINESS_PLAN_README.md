# 事業計画書（CSV）

## ファイル

- **business_plan.csv** … 重要事項だけをまとめた1ファイル（判定・事業・財務・条件・リスク・ロードマップ）

## 開き方

- **Excel**: ダブルクリック、または「データ」→「テキスト/CSVから」でUTF-8指定
- **Googleスプレッドシート**: アップロード → ダブルクリック → 「スプレッドシートで開く」

## その他の数値・タスク

- 月次財務: `10_financial_plan.csv`
- KPI: `10_kpi_dashboard.csv`
- タスク: `10_task_management.csv`
